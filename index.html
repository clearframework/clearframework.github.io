CLEAR Framework

Community-driven Lightweight Enterprise Architecture Framework

Quick Start →

Community-driven

Knowledge about Enterprise Architecture is spread across our industry, there is no better way to compile it than doing it collaboratively.

Lightweight

Framework to be actionable has to be lean. As a user you have to able to grasp it in short period of time. It should consist of well chosen and practical set of tools.

Easy to learn

We want our framework to be widely adapted, this is why we think we should strive for simplicity and concepts that can easily understood.

Accessible

We strongly believe that current approach to Enterprise Architecture is wrong and lack of accessibility and open standards blocks it's progress.

Rapid to implement

You should be able to include building blocks from framework from day one. It should not be all or nothing.
